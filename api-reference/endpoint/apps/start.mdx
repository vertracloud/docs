---
title: 'Start Application'
sidebarTitle: 'Start Application'
api: 'POST https://api.vertracloud.app/v1/apps/{id}/start'
authMethod: 'bearer'
description: 'Starts an application by its ID.'
---

<ParamField path="id" type="string" required>
  The ID of the application to start. If the application
  belongs to an organization, include the organization ID in the format `appId-orgId`.
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. `success` if successful, `error` if
  not.
</ResponseField>

<ResponseExample>

```json Response
{
  "status": "success"
}
```

</ResponseExample>
