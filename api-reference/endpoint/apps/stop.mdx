---
title: 'Stop Application'
sidebarTitle: 'Stop App'
api: 'POST https://api.vertracloud.app/v1/apps/{id}/stop'
authMethod: 'bearer'
description: 'Stops an application by its ID.'
---

<ParamField path="id" type="string" required>
  The ID of the application to stop.
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. `success` if successful, `error` if
  not.
</ResponseField>

<ResponseExample>

```json Response
{
  "status": "success"
}
```

</ResponseExample>
