---
title: 'Get Recent Deployments'
sidebarTitle: 'Recent Deployments'
api: 'GET https://api.vertracloud.app/v1/apps/{app_id}/deploys/recent'
authMethod: 'bearer'
description: 'Retrieves the 20 most recent deployments for a specific application.'
---

<ParamField path="app_id" type="string" required>
The ID of the application whose recent deployments will be retrieved.
</ParamField>

### Response

<ResponseField name="status" type="string">
Indicates if the request was successful. Returns `success` on success.
</ResponseField>

<ResponseField name="response" type="array">
An array of deployment objects.

<Expandable title="Toggle object">
  <ResponseField name="app_id" type="string">
  The ID of the application.
  </ResponseField>

  <ResponseField name="commit_id" type="string">
  The commit ID of the deployment.
  </ResponseField>

  <ResponseField name="message" type="string">
  Commit message.
  </ResponseField>

  <ResponseField name="pusher" type="string">
  The username of the person who triggered the deployment.
  </ResponseField>

  <ResponseField name="branch" type="string">
  The branch that was deployed.
  </ResponseField>

  <ResponseField name="created_at" type="string">
  ISO timestamp of when the deployment occurred.
  </ResponseField>
</Expandable>
</ResponseField>

<ResponseExample>
```json
{
  "status": "success",
  "response": [
    {
      "app_id": "123",
      "commit_id": "abc123",
      "message": "Fix deployment bug",
      "pusher": "user1",
      "branch": "main",
      "created_at": "2025-08-14T20:00:00.000Z"
    }
  ]
}
```
</ResponseExample>

### Error Responses

<ResponseField name="status" type="string">
Returns `error` if the request fails.
</ResponseField>

<ResponseField name="code" type="string">
Possible error codes:
- `UNAUTHORIZED`: User is not authenticated or lacks permissions.
- `INTEGRATION_NOT_FOUND`: Application or deployments not found.
</ResponseField>

<ResponseExample>
```json
{
  "status": "error",
  "code": "INTEGRATION_NOT_FOUND"
}
```
</ResponseExample>
