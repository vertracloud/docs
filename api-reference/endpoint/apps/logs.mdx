---
title: 'Application Logs'
sidebarTitle: 'Application Logs'
api: 'GET https://api.vertracloud.app/v1/apps/{id}/logs'
authMethod: 'bearer'
description: 'Retrieves the logs for an application by its ID.'
---

<ParamField path="id" type="string" required>
  The ID of the application to retrieve realtime logs for. If the application
  belongs to an organization, include the organization ID in the format
  `appId-orgId`.
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. `success` if successful, `error` if
  not.
</ResponseField>

<ResponseField name="response" type="string">
  The log content for the specified application.
</ResponseField>

<ResponseExample>

```json Response
{
  "status": "success",
  "response": "2025-08-14T19:58:00Z [INFO] Application started successfully\n2025-08-14T19:58:01Z [DEBUG] Connected to database\n2025-08-14T19:58:02Z [ERROR] Failed to process request"
}
```

</ResponseExample>
