---
title: 'Start Database'
sidebarTitle: 'Start Database'
api: 'POST https://api.vertracloud.app/v1/databases/{id}/start'
authMethod: 'bearer'
description: 'Starts an database by its ID.'
---

<ParamField path="id" type="string" required>
  The ID of the database to start. If the database
  belongs to an organization, include the organization ID in the format `dbId-orgId`.
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. `success` if successful, `error` if
  not.
</ResponseField>

<ResponseExample>

```json Response
{
  "status": "success"
}
```

</ResponseExample>
