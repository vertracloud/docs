{/* @see success — @vertracloud/api-types */}
<ParamField path="id" type="string" required>
  The ID of the database to restart. If the database
  belongs to an workspace, include the workspace_id in the format `dbId-workspace_id`.
</ParamField>

### Response

<ResponseField name="response" type="string">
  The response message, indicating successful operation.
</ResponseField>

<ResponseExample>

```json Response
{
  "response": "success"
}
```

</ResponseExample>
