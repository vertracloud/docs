---
title: 'Reset Database Password'
sidebarTitle: 'Reset Password'
api: 'POST https://api.vertracloud.app/v1/databases/{id}/credentials/reset'
authMethod: 'bearer'
description: 'Resets the password for a specific database by its ID.'
---

<ParamField path="id" type="string" required>
  The ID of the database to reset the password for.
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. `success` if successful, `error` if not.
</ResponseField>

<ResponseField name="response" type="string">
  The new password for the database.
</ResponseField>

<ResponseExample>

```json Response
{
  "status": "success",
  "response": "newStrongPassword123"
}
```

</ResponseExample>

### Error Responses

<ResponseField name="status" type="string">
  Indicates whether the call was successful. `error` if not successful.
</ResponseField>

<ResponseField name="code" type="string">
  The error code indicating the reason for failure. Possible values:
  - `UNAUTHORIZED`: The user is not authenticated or lacks sufficient permissions.
  - `DATABASE_NOT_FOUND_OR_FORBIDDEN`: The specified database does not exist or the user lacks access.
  - `INVALID_PLAN_CONFIG`: The database plan configuration is invalid.
  - `FAILED_TO_REBUILD_DATABASE`: Failed to rebuild the database with the new password.
</ResponseField>

<ResponseExample>

```json Error Response (403)
{
  "status": "error",
  "code": "UNAUTHORIZED"
}
```

```json Error Response (404)
{
  "status": "error",
  "code": "DATABASE_NOT_FOUND_OR_FORBIDDEN"
}
```

```json Error Response (500)
{
  "status": "error",
  "code": "FAILED_TO_REBUILD_DATABASE"
}
```

</ResponseExample>
