title: 'Check Infrastructure Status'
sidebarTitle: 'Check Status'
api: 'GET https://api.vertracloud.app/v1/status'
authMethod: 'none'
description: 'Checks if the infrastructure is fully operational.'
---

### Response

<ResponseField name="status" type="string">
  Indicates the current infrastructure status. Possible values: `healthy`,
  `unknown`, `degraded`.
</ResponseField>

<ResponseField name="response" type="string">
  A message describing the infrastructure status. Possible values: `All systems
  operational`, `Some systems are degraded`, `Some systems are down`.
</ResponseField>

<ResponseExample>

```json Response
{
  "status": "healthy",
  "response": "All systems operational"
}
```

</ResponseExample>
