---
title: 'Remove Application from Organization'
sidebarTitle: 'Remove App from Org'
api: 'DELETE https://api.vertracloud.app/v1/organizations/{org_id}/apps/{app_id}'
authMethod: 'bearer'
description: 'Removes an application from an organization by specifying the organization and application IDs.'
---

<ParamField path="org_id" type="string" required>
  The ID of the organization to remove the application from.
</ParamField>

<ParamField path="app_id" type="string" required>
  The ID of the application to remove from the organization.
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. `success` if successful, `error` if not.
</ResponseField>

<ResponseExample>

```json Response
{
  "status": "success"
}
```

</ResponseExample>

### Error Responses

<ResponseField name="status" type="string">
  Indicates whether the call was successful. `error` if not successful.

<ResponseField name="code" type="string">
  The error code indicating the reason for failure. Possible values:
  - `FORBIDDEN`: The user is not authenticated, lacks sufficient permissions, or is not an owner/admin of the organization.
  - `APP_NOT_FOUND`: The specified application does not exist or does not belong to the user.
</ResponseField>
