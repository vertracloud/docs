---
title: Add Application to Workspace
sidebarTitle: Add App to Org
api: POST https://api.vertracloud.app/v1/workspaces/{org_id}/apps/{app_id}
authMethod: bearer
description: Adds an application to an workspace by specifying the workspace and application IDs.
---

{/* @see success — @vertracloud/api-types */}
<ParamField path="org_id" type="string" required>
  The ID of the workspace to add the application to.
</ParamField>

<ParamField path="app_id" type="string" required>
  The ID of the application to add to the workspace.
</ParamField>

### Response

<ResponseExample>

```json Response
{}
```

</ResponseExample>

### Error Responses

<ResponseField name="code" type="string">
  The error code indicating the reason for failure. Possible values:
  - `FORBIDDEN`: The user is not authenticated, lacks sufficient permissions, or is not an owner/admin of the workspace.
  - `APP_NOT_FOUND`: The specified application does not exist or does not belong to the user.
</ResponseField>

<ResponseField name="message" type="string">
  A descriptive message providing additional details about the error.
</ResponseField>

<ResponseExample>

```json Error Response (403)
{
  "code": "FORBIDDEN"
}
```

```json Error Response (404)
{
  "code": "APP_NOT_FOUND"
}
```

</ResponseExample>
