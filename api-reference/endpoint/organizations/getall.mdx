---
title: 'Get User Organizations'
sidebarTitle: 'User Organizations'
api: 'GET https://api.vertracloud.app/v1/organizations/all/{id}'
authMethod: 'bearer'
description: 'Retrieves all organizations a user is a member of.'
---

<ParamField path="id" type="string" required>
  The ID of the user whose organizations are to be retrieved. Use `@me` to refer to the authenticated user.
</ParamField>

### Error Responses

<ResponseField name="status" type="string">
  Indicates whether the call was successful. `error` if not successful.
</ResponseField>

<ResponseField name="code" type="string">
  The error code indicating the reason for failure. Possible values:
  - `UNAUTHORIZED`: The user is not authenticated.
  - `USER_NOT_FOUND`: The specified user does not exist.
  - `ID_MUST_BE_A_STRING`: The provided user ID is not a string.
</ResponseField>

<ResponseExample>

```json Error Response (404)
  "code": "USER_NOT_FOUND"
}
```

</ResponseExample>
