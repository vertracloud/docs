{/* @see success — @vertracloud/api-types */}
<ParamField path="id" type="string" required>
  The ID of the workspace to delete.
</ParamField>

### Response

<ResponseField name="message" type="string">
  A message indicating the result of the operation (e.g., `ORGANIZATION_DELETED`).
</ResponseField>

<ResponseExample>

```json Response
{
  "message": "ORGANIZATION_DELETED"
}
```

</ResponseExample>

### Error Responses

<ResponseField name="code" type="string">
  The error code indicating the reason for failure. Possible values:
  - `UNAUTHORIZED`: The user is not authenticated or lacks sufficient permissions.
  - `ORGANIZATION_NOT_FOUND`: The specified workspace does not exist or the user is not the owner.
</ResponseField>

<ResponseField name="message" type="string">
  A descriptive message providing additional details about the error.
</ResponseField>

<ResponseExample>

```json Error Response (403)
{
  "code": "UNAUTHORIZED"
}
```

```json Error Response (404)
{
  "code": "ORGANIZATION_NOT_FOUND"
}
```

</ResponseExample>
