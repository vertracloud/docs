---
title: 'Add Database to Organization'
sidebarTitle: 'Add Database'
api: 'POST https://api.vertracloud.app/v1/organizations/{org_id}/databases/{db_id}'
authMethod: 'bearer'
description: 'Adds a database to an organization by its ID.'
---

<ParamField path="org_id" type="string" required>
  The ID of the organization to add the database to.
</ParamField>

<ParamField path="db_id" type="string" required>
  The ID of the database to add to the organization.
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. `success` if successful, `error` if not.
</ResponseField>

<ResponseExample>

```json Response
{
  "status": "success"
}
```

</ResponseExample>

### Error Responses

<ResponseField name="status" type="string">
  Indicates whether the call was successful. `error` if not successful.
</ResponseField>

<ResponseField name="code" type="string">
  The error code indicating the reason for failure. Possible values:
  - `FORBIDDEN`: The user is not authenticated or lacks sufficient permissions (requires `owner` or `admin` role).
  - `DATABASE_NOT_FOUND`: The specified database does not exist or the user is not the owner.
</ResponseField>

<ResponseExample>

```json Error Response (403)
{
  "status": "error",
  "code": "FORBIDDEN"
}
```

```json Error Response (404)
{
  "status": "error",
  "code": "DATABASE_NOT_FOUND"
}
```

</ResponseExample>
