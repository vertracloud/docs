## Overview

July marked a major milestone with the launch of the payment system, organization management, and comprehensive application settings. The platform evolved from a basic hosting service to a full-featured application management system with team collaboration and billing capabilities.

---

## New Features

### App Settings Tabs
Organized application management across dedicated tabs.

- **General** — Basic app settings and information
- **Environment Variables** — Secrets and configuration management
- **Deploys** — Deployment history and auto-deploy setup
- **Network** — Custom domain and networking options
- **Danger Zone** — Destructive operations like app deletion

### Environment Variables
Complete secrets and configuration management system.

- Full CRUD operations for environment variables
- Dedicated secrets section for sensitive data
- Bulk paste support for quick setup
- Real-time environment variable injection
- Variable encryption at rest

### Custom Domains
DNS and subdomain management for applications.

- Configure custom domain for any application
- DNS settings guidance and validation
- Subdomain customization
- SSL/TLS certificate provisioning

### Auto Deploys
Automatic deployment from GitHub webhooks.

- GitHub webhook integration
- Automatic deployment on push to specified branch
- Webhook configuration and management
- Deployment logs and history

### Organizations
Full team management with role-based access control.

- Create and manage organizations
- Member management with role assignment
- Commit permissions per role
- Backups access control per role
- Organization deletion

### Pricing Page
Complete pricing structure with plan comparison.

- Tier comparison table
- Feature list per plan
- Database plan pricing
- Annual pricing options
- Clear plan differentiation

### Payment System
Mercado Pago integration for billing.

- **PIX Integration** — Brazilian instant payment method
- Invoice page showing transaction history
- Pay page for plan purchases
- Plan renewal management
- Multiple payment options support

### Redeem Codes
Code-based account credit system.

- Redeem promotional codes
- Code validation and verification
- Credit application to account
- Usage tracking and history

### Plan Management
User-friendly plan management interface.

- Upgrade to higher tier
- Downgrade to lower tier
- Plan renewal
- Plan change history

### Plan Expiry Notifications
Alerts for upcoming plan expiration.

- Email notification before expiration
- Dashboard warning indicators
- Time to renew information
- Renewal prompt in settings

### API Key
Request and manage API keys from dashboard.

- Generate API keys
- Revoke keys
- Key management interface
- API key usage in applications

### Activities and Audit Logs
Complete activity tracking for accountability.

- Activity tracking for applications
- Database activity logs
- User action history
- Timestamp and details for each action
- Export capability
