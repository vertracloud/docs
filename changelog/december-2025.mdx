## Overview

December brought new language runtime support and improved the promotional system with the holiday VERTRA30 coupon. Email delivery improvements ensure better communication with users across the platform.

---

## New Features

### Go Language Support
Added Go runtime for high-performance applications.

- Go language support for deployment
- Automatic Go compilation
- Go module dependency management
- Fast build times
- Ideal for microservices and CLI tools

### PHP Language Support
Added PHP 8.x runtime for web applications.

- PHP 8.x runtime support
- Common PHP web framework compatibility
- Apache/Nginx configuration support
- PHP package management via Composer
- Web application hosting

### Coupon System
Promotional code support for discounts on orders.
