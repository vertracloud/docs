## Overview

January 2026 marked the official public beta launch of Vertra Cloud with a full-featured REST API and comprehensive documentation. The platform opened to the community with team collaboration features and multi-region infrastructure support.

---

## Features

### API v1
The first public version of the Vertra Cloud RESTful API.

- Comprehensive REST API for all platform operations
- Full API reference documentation
- Request/response examples
- API key authentication
- Rate limiting and quota management

### Workspaces
Team collaboration with role-based access control.

- Create and manage workspaces for teams
- 5 predefined roles: Owner, Admin, Developer, Operator, Viewer
- Role-based permission management
- Member invitation and management
- Workspace isolation and security

### Public Beta
Official open beta period launch.

- Platform opened for public user registration
- Beta testing and feedback collection
- Community user support
- Stability and performance improvements based on feedback

### Documentation
Complete API and platform documentation.

- Docs site at docs.vertracloud.app
- API reference with all endpoints
- Code examples and tutorials
- Getting started guides
- FAQ and troubleshooting
