## Overview

February 2026 brought critical database management features and integrated development tools. The official VSCode extension, managed database support, and enhanced security features make Vertra Cloud a comprehensive platform for full-stack development.

---

## New Features

### VSCode Extension
Official Vertra Cloud extension for VS Code.

- Direct deployment from the editor
- Application management from VS Code
- Real-time log viewing in sidebar
- Quick status and metrics display
- Integrated project creation
- Available on VS Code Marketplace

### Managed Databases
Production-ready database hosting with automatic management.

- **PostgreSQL 17** — Latest PostgreSQL with automatic updates
- **MongoDB 8.0** — Latest MongoDB with replica sets
- Automatic daily backups
- Point-in-time recovery
- Automated patch management
- Read replicas for scaling

### Automatic TLS Certificates
Encrypted database connections out of the box.

- Automatic TLS certificate generation
- Certificate renewal management
- Client certificate support
- Secure connection requirements
- Certificate pinning support

### MySQL Support
MySQL 8.0 database hosting.

- MySQL 8.0 with latest features
- Mandatory X509 authentication for security
- Connection pooling
- Automatic backups
- Binary log management

### Network Protection
Enhanced security and traffic management.

- Automatic traffic monitoring
- DDoS mitigation
- Rate limiting per application
- Connection throttling
- IP whitelisting support
