## Overview

May marks the beginning of Vertra Cloud with the initial dashboard launch. This foundational release provides core application management, database connectivity, and the building blocks for all subsequent features.

---

## Features

### Dashboard
Initial dashboard page for application and database management.

- View all deployed applications
- View all connected databases
- Quick status overview
- Project listing and filtering
- Basic resource management

### Database Details
Connected database management interface.

- View database information
- Connection credentials display
- Database usage statistics
- Configuration management
- API integration

### App Details
Application management and file viewing.

- Application overview
- File manager integration
- Tab navigation from URL parameters
- Application logs
- Deployment status

### Apps and Databases Pages
Dedicated listing pages for organization.

- Apps page showing all applications
- Databases page showing all databases
- Filtering and sorting
- Quick actions per resource

### Custom Domain Field
Initial custom domain support in application model.

- Custom domain configuration on applications
- Domain validation
- DNS setup guidance
- Foundation for custom domain features
