## What is Vertra Cloud?

Vertra Cloud is a cloud hosting platform that allows you to deploy, manage and scale applications and databases in a simple and fast way. The platform takes care of all the infrastructure so you can focus on what matters: your code.

## Available Databases

<CardGroup cols={2}>
  <Card title="PostgreSQL" icon="elephant">
    **Version:** 17 &nbsp;|&nbsp; **Port:** 5432

    Relational database with SQL, JSONB and extensions support. Mandatory TLS with automatic certificates.
  </Card>
  <Card title="MongoDB" icon="leaf">
    **Version:** 8.0.11 &nbsp;|&nbsp; **Port:** 27017

    NoSQL document-oriented database. Mandatory TLS with platform-signed CA.
  </Card>
  <Card title="Redis" icon="bolt">
    **Version:** 7 (Bookworm) &nbsp;|&nbsp; **Port:** 6379

    High-performance in-memory store for caching and pub/sub. Mandatory TLS.
  </Card>
  <Card title="MySQL" icon="database">
    **Version:** 8.0 &nbsp;|&nbsp; **Port:** 3306

    Relational database with mandatory X509 authentication and auto-generated TLS certificates.
  </Card>
</CardGroup>
